# Bottom 100
The bottom 100 was a benchmark developed by [Babelcorp](./Babelcorp). It was designed to scientifically determine exactly how worthless a piece of hardware is.

## The Results

| Hardware | Score | Submitted By |
|----------|-------|--------------|
| Intel Compute Stick| 0.8 Tokens/s | Babelcorp Management |

## Benchmark Something

The benchmark uses Ollama. Install Ollama on the machine, and run `ollama pull llama3.2:1b` to pull the right model. 

You can run the benchmark by running [The Worker Code of Babelcom](https://github.com/johncave/internet-of-babel) with the benchmark flag. Eg:
```bash
cd worker
go run benchmark
```

Edit the Librarian's Wiki to submit your score. 

Only facetious benchmarks with a link will be accepted.
